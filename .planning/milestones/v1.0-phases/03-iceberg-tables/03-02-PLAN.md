Glue database: open_images
Athena workgroup: open-images (engine v3, 10GB scan limit)

Two-step pattern per table:
1. DROP TABLE IF EXISTS open_images.{iceberg_table}
2. DROP TABLE IF EXISTS open_images.raw_{table}
3. CREATE EXTERNAL TABLE open_images.raw_{table} (all STRING columns, OpenCSVSerDe)
4. CREATE TABLE open_images.{table} WITH (table_type='ICEBERG', ...) AS SELECT with CAST

Critical rules:
- OpenCSVSerDe: ALL columns must be STRING (types are ignored)
- Iceberg CTAS: table_type='ICEBERG', is_external=false, format='PARQUET', write_compression='SNAPPY'
- Boolean casting: CASE WHEN col = '1' THEN true ELSE false END (NOT CAST)
- __BUCKET__ placeholder for S3 paths (runner substitutes at execution time)
- Statements separated by semicolons
- skip.header.line.count = '1' for CSVs with headers
